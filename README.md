# Heleno Lopes Fotógrafo - Landing Page

Esta é uma landing page profissional e portfólio criativo desenvolvida para o fotógrafo Heleno Lopes.

## 🚀 Tecnologias Utilizadas

- **React 19** com **TypeScript**
- **Vite** como ferramenta de build
- **Tailwind CSS 4** para estilização moderna e responsiva
- **Motion (Framer Motion)** para animações suaves
- **Lucide React** para ícones consistentes

## 🛠️ Como Executar o Projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Para gerar a versão de produção:
   ```bash
   npm run build
   ```

## 📄 Estrutura do Projeto

- `src/components/`: Componentes modulares da interface.
- `src/constants.ts`: Dados mockados e links globais.
- `src/types.ts`: Definições de tipos TypeScript.
- `src/index.css`: Configuração do tema Tailwind 4.

## ✨ Funcionalidades

- Design 100% responsivo.
- Galeria de portfólio com layout assimétrico e modal de visualização.
- Botão flutuante de WhatsApp para conversão rápida.
- Seções institucionais, serviços, depoimentos e valores.
- Otimizado para SEO com meta tags e semântica HTML5.
